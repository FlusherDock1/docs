# Image

`image` - displays an image column with the option to resize the output.

```yaml
avatar:
    label: Avatar
    type: image
    sortable: false
    width: 150
    height: 150
    options:
        quality: 80
```

See the [image resizing article](../services/resizer.md#oc-resize-parameters) for more information on what options are supported.
